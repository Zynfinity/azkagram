## Build App Release

1. Clone Repo
```bash
git clone https://github.com/azkadev/azkagram.git
cd azkagram
flutter pub get
```

2. Build app

    - Android
        ```
        flutter build apk --release
        ``` 
    - iOS
        ```
        flutter build ios --release
        ``` 
    - Linux
        ```
        flutter build linux --release
        ``` 
    - macOS
        ```
        flutter build macos --release
        ``` 
    - Windows
        ```
        flutter build windows --release
        ```
